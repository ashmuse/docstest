Premium unlocks the full platform — all models, LoRAs, Styles, InPaint, Private Generation, and video generation. See [/premium](/premium) for current pricing and plan options.

## Free tier vs Premium

| Feature | Free | Premium |
|---|---|---|
| Models | Realism, Anime | All models |
| Generations per day | 5 | Unlimited |
| Video generation | View only | ✓ |
| Token purchases | — | ✓ |
| LoRAs | — | ✓ |
| Styles | — | ✓ |
| Better Quality | — | ✓ |
| InPaint | — | ✓ |
| Private Generation | — | ✓ |

## How to upgrade

Go to [/premium](/premium) and select a plan. Payment by card or cryptocurrency is accepted. See [Crypto Payments](/account/crypto-payments).

Premium activates immediately after payment.

## Token-based generation

Premium generation runs on tokens. Each generation type has a token cost — see [/premium](/premium) for current rates. Tokens are purchased separately from the subscription and don't expire.

See [Tokens](/account/tokens) for how to check your balance and buy more.
